POLÍTICA DE PRIVACIDAD

HVU – Hogar de Varones Universitarios

Última actualización: Febrero 2026

1. Introducción

La aplicación HVU – Hogar de Varones Universitarios (en adelante, "la Aplicación") es una plataforma institucional diseñada para facilitar la consulta de información relevante para los estudiantes residentes del Hogar de Varones Universitarios.

Nos comprometemos a proteger la privacidad y seguridad de los datos personales de nuestros usuarios, conforme a la legislación aplicable en materia de protección de datos personales.

2. Información que recopilamos

La Aplicación puede recopilar los siguientes datos:

  • Nombre completo
  • Correo electrónico institucional
  • Matrícula o identificador académico
  • Información relacionada con el estatus del estudiante dentro del sistema
  • Datos técnicos básicos (como tipo de dispositivo o versión de sistema operativo)

La Aplicación no recopila datos sensibles, financieros ni biométricos.

3. Finalidad del uso de los datos

Los datos recopilados se utilizan exclusivamente para:

  • Permitir el acceso seguro a la Aplicación
  • Mostrar información personalizada al estudiante
  • Gestionar módulos internos (avisos, reportes, estadísticas, información administrativa)
  • Mejorar el funcionamiento y seguridad del sistema

Los datos no se utilizan para fines comerciales ni publicitarios.

4. Publicidad

La Aplicación no contiene anuncios ni utiliza servicios de publicidad de terceros.

5. Compartición de información

La información recopilada:

  • No se vende
  • No se alquila
  • No se comparte con terceros externos

Los datos solo son accesibles por el personal administrativo autorizado del Hogar de Varones Universitarios, cuando sea necesario para fines institucionales.

6. Seguridad de la información

Implementamos medidas técnicas razonables para proteger la información, incluyendo:

  • Transmisión de datos cifrada (HTTPS)
  • Acceso restringido según roles (estudiante, administrador)
  • Autenticación mediante credenciales seguras

7. Derechos del usuario

El usuario puede:

  • Solicitar la actualización de sus datos
  • Solicitar la eliminación de su cuenta (cuando aplique)
  • Solicitar información sobre el tratamiento de sus datos

Para ejercer estos derechos puede contactar al correo institucional indicado en esta política.

8. Retención de datos

Los datos se conservan únicamente durante el tiempo necesario para cumplir con las funciones académicas y administrativas de la institución.

9. Cambios a esta política

Nos reservamos el derecho de modificar esta Política de Privacidad en cualquier momento. Cualquier cambio será publicado dentro de la Aplicación o en el sitio web oficial.
